# Maharashtra Construction Demand Analysis
## Project Overview

This project analyzes construction material demand across Maharashtra to uncover regional consumption patterns, pricing trends, and supply-demand concentration.

Using SQL for data transformation and aggregation, and Tableau for dashboard visualization, the analysis identifies actionable insights for distributors, suppliers, and strategic planners in the construction ecosystem.

The study focuses on:

District-wise demand concentration

Material category performance

Urban vs semi-urban consumption trends

Seasonal and pricing dynamics

## Business Objective

To derive data-driven insights that can support:

Inventory optimization

Regional expansion strategy

Pricing and margin decisions

Channel allocation planning

## Tools & Technologies

SQL (Data Cleaning & Aggregation)

Excel (Data Validation & Transformation)

Tableau (Interactive Dashboard Development)

Exploratory Data Analysis (EDA)
